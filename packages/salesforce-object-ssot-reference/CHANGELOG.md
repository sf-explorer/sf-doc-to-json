# Changelog

All notable changes to @sf-explorer/salesforce-object-ssot-reference will be documented in this file.

## [1.0.0] - 2024-12-10

### Added
- Initial release as separate package
- SSOT objects from DMO APIs
- Lightweight API for object descriptions
- Full TypeScript support
- Browser and Node.js support

### Features
- `loadIndex()` - Load DMO index
- `getObject()` - Get full object details
- `searchObjects()` - Search by name
- `getObjectDescription()` - Get lightweight metadata
- `searchObjectsByDescription()` - Search by description
- `loadAllObjects()` - Load all SSOT objects
- Caching support
