To test this properly:
1. Use Vite: npm create vite@latest my-test && cd my-test && npm install
2. Install package: npm install @sf-explorer/salesforce-object-reference
3. Import in your code: import { loadAllDescriptions } from '@sf-explorer/salesforce-object-reference'
4. Run: npm run dev

üìù Example Code

main.js 
import { 
  loadAllDescriptions,
  getObjectDescription,
  searchObjectsByDescription 
} from '@sf-explorer/salesforce-object-reference';

// Load all descriptions
const descriptions = await loadAllDescriptions();
console.log(`Loaded ${Object.keys(descriptions).length} objects`);

// Get specific object
const account = await getObjectDescription('Account');
console.log(account.description);
console.log(`${account.fieldCount} fields`);

// Search
const results = await searchObjectsByDescription('invoice');
console.log(`Found ${results.length} objects`);
